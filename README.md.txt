# Telegram Schedule Bot

Телеграм-бот для просмотра расписания 📚

## Возможности

- Просмотр расписания по дням недели
- Парсинг HTML файла
- Статистика пользователей
- Рассылка сообщений

## Технологии

- Python
- Django
- BeautifulSoup
- python-telegram-bot

## Запуск проекта

Установка библиотек:

pip install -r requirements.txt

Запуск бота:

python bot.py
